Полноценный дашборд аналитики с авторизацией, real-time данными и адаптивной версткой. Разработан как тестовое задание для демонстрации навыков Fullstack-разработки. Frontend выполнен на VUE, backend на C#.

## ✨ Функционал

- 🔐 JWT-авторизация с валидацией токена при старте
- 📈 Интерактивные карточки статистики с круговыми прогресс-барами (SVG)
- 📋 Фильтрация задач по статусу и сортировка по приоритету (High/Medium/Low)
- 📅 Календарь событий с навигацией по месяцам
- 🔥 Лента активности команды с группировкой по типам действий
- 🔄 PHP-прослойка для агрегации данных и группировки JSON

## Как запустить

### 1. Backend (.NET)

- cd backend
- dotnet restore
- dotnet run

## API доступен на http://localhost:5102

### 2. Frontend (Vue 3)

- cd frontend
- npm install
- npm run dev

# Приложение откроется на http://localhost:3000

### PHP Proxy (опционально)

php proxy.php

# Выводит агрегированный JSON для внешних интеграций
